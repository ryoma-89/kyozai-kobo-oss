# 数式入力のルール（MathGraph PDF Studio）

左パネルの「式を追加」に入力できる式の書き方をまとめます。

---

## 1. 使える変数・定数

| 種類 | 記号 |
|---|---|
| 座標平面の変数 | `x`, `y` |
| 媒介変数（`(…, …)` の中だけ） | `t` |
| 円周率 | `pi`（`π` も可） |
| ネイピア数 | `e` |
| 円周率×2 | `tau` |

- 変数は **`x` と `y`（媒介変数は `t`）だけ**です。`a` などは使えません。
- 大文字 `PI` `E` も使えます。

## 2. 使える演算子

`+` `-` `*` `/` `^`（べき乗）、丸括弧 `( )`

- **かけ算の `*` は省略できます**：`2x`、`3(x+1)`、`2sin(x)` はそれぞれ `2*x`、`3*(x+1)`、`2*sin(x)` と同じ。
- べき乗は `^`：`x^2`、`x^(n+1)`。

## 3. 使える関数

```
sqrt  abs  cbrt  nthRoot        （平方根・絶対値・立方根・n乗根）
sin  cos  tan                   （三角関数、引数はラジアン）
asin acos atan                  （逆三角関数）
sinh cosh tanh                  （双曲線関数）
exp  log  ln  log10  log2       （指数・対数）
floor ceil round sign           （床・天井・四捨五入・符号）
min  max  pow                   （最小・最大・べき）
```

- **`log(x)` と `ln(x)` はどちらも自然対数**です。常用対数（底10）は `log10(x)`。
- 絶対値は `abs(x)` または **`|x|`**（縦棒）でも書けます。
- 三角関数の角度はラジアンです（`sin(pi/2)` = 1）。

### 根号（√）とべき乗根
- **`√` は括弧なしでも使えます**：`√x`＝`sqrt(x)`、`√(x+1)`、`2√x`＝`2*sqrt(x)`、`√sin(x)`。
- **入れ子も可**：`√√x`＝`sqrt(sqrt(x))`。
- `√` は直後の1つのかたまり（数・変数・関数・括弧）にかかります。`√x^2` は `(√x)^2` の意味なので、`√(x^2)` としたいときは括弧を付けてください。
- **べき乗根**：`x^(1/3)` は立方根で、**負の数でも実数の値**になります（`(-8)^(1/3) = -2`）。奇数乗根も同様。`cbrt(x)`（立方根）や `nthRoot(x, n)`（n乗根）でも書けます。

## 4. ガウス記号（床関数）

- `[x]` は `floor(x)`（ガウス記号）として解釈します。例: `y = [x]`、`y = [x/2]`。
- 入れ子も可：`[[x]/2]`。前に数や変数があるかけ算も可：`2[x]`。
- `⌊x⌋`（床）、`⌈x⌉`（天井）も使えます。

## 5. 式の形（グラフの種類）

### (1) y = f(x) 型（xの関数）
```
y = x^2 - 4*x + 3      y = abs(x)        y = sqrt(x + 2)
y = 1/(x - 1)          y = sin(x)        y = log(x)
y = exp(x)             x^2               ← 「y =」は省略可
```

### (2) x = f(y) 型（yの関数）／縦線
```
x = y^2       x = sin(y)      x = 2y + 1      y^2 = x
x = 3         ← 定数なら縦線
```

### (3) 陰関数 f(x,y) = 0
```
x^2 + y^2 = 9          x^2/4 + y^2/9 = 1
x*y = 1                sin(x) + cos(y) = 0
```

### (4) 媒介変数表示 (x(t), y(t))
```
(cos(t), sin(t))          円
(t, t^2)                  放物線
(2cos(t), 3sin(t))        楕円
(t - sin(t), 1 - cos(t))  サイクロイド
(cos(3t), sin(2t))        リサージュ曲線
```
- パラメータは **`t`** を使います。
- **t の範囲**は式カードの「t：〇〜〇」欄で指定します（既定は 0〜2π、「2π」ボタンで素早く設定）。

## 6. 不等式（領域の塗りつぶし）

不等号 `<` `<=` `>` `>=` を使うと、その領域を塗りつぶします（`=` は等式＝曲線）。

```
y >= x^2 - 4*x + 3     y <= -x + 2      x >= -1        x < 3
x^2 + y^2 <= 9         x + y < 2        x >= y^2 - 1
```

- 境界線は既定で実線です（式ごとに破線へ変更可）。
- `=<` `=>`、全角の `≦` `≧` も `<=` `>=` として解釈します。

### 範囲指定（連鎖不等式）
```
1 <= x <= 3        0 <= y <= x^3        -1 < y < 1
```
- `A <= B <= C` は「A ≤ B かつ B ≤ C」の共通領域になります。
- 不等号の向きはそろえてください（`1 <= x >= 0` はエラー）。1つの式で不等号は最大2つまで。

### かつ・または（and / or）
複数の不等式を `and`（かつ）・`or`（または）で結合し、1つの領域にできます。
```
y >= 0 and y <= x^2
(x >= 0 and y <= x^2) or (x < 0 and y <= -x)
x^2 + y^2 <= 9 or x^2 + y^2 >= 16
```
- `かつ` `または` `&&` `||` `∧` `∨` も同じ意味です。丸括弧でグループ化できます。
- 領域として結合できるのは**不等式のみ**です。`and` は `or` より強く結びつきます。
- ただし、等式の曲線を1つだけ含む `and` は、次の「曲線を不等式で切り取る」用途として使えます。

### 曲線を不等式で切り取る

等式の曲線を1つだけ含む `and` では、不等式条件を満たす部分だけ曲線を描けます。

```
y = x^2 and x >= 0
x^2 + y^2 = 9 and y >= 0
(cos(t), sin(t)) and x >= 0
y = sin(x) and 0 <= x <= pi
```

- 曲線は1つだけ指定できます。
- `and` の残りは不等式条件として扱われます。
- この場合は領域を塗るのではなく、曲線の表示部分だけを切り取ります。

## 7. LaTeX 記法での入力

LaTeX をそのまま貼り付けても解釈します。

```
y = \frac{x^2}{4}          →  y = x^2/4
y = \sqrt{x+2}             →  y = sqrt(x+2)
y = \sqrt[3]{x}            →  y = nthRoot(x, 3)
x^2 + y^2 \leq 9           →  x^2 + y^2 <= 9
y = \sin(x) + \frac{1}{2}
y = |x|
```

対応コマンド: `\frac{}{}` `\dfrac` `\sqrt{}` `\sqrt[n]{}` `x^{…}` `\cdot` `\times` `\div` `\le` `\leq` `\ge` `\geq` `\pi` `\sin`〜`\ln`〜`\exp` `\left(` `\right)` `\lvert…\rvert`（絶対値） `\land`/`\lor`（かつ/または）など。

## 8. その他の便利機能

- **全角対応**：`ｙ＝ｘ＾２－１` のような全角入力も解釈します。
- **点の座標入力**：点を追加するときの x座標・y座標には、整数だけでなく `1/2`、`0.25`、`pi`、`pi/2`、`e`、`sqrt(2)`、`\frac{1}{2}` なども入力できます。確定すると数値に変換されます。
- **点から軸への垂線**：追加した点ごとに、「x軸へ垂線」「y軸へ垂線」を個別にオン/オフできます。
- **数式ラベル**：グラフ上に置く数式ラベルは LaTeX 記法で入力します（`b=(a+1)^2-1` など）。
- **交点の自動検出**：右パネルで有効化すると、表示中の曲線どうしの交点を自動表示します。
- **グラフの縦横比**：右パネルの表示設定で、座標単位を正方形に保ったまま、1:1・4:3・16:9などの指定比率にできます。指定比率ではグラフを引き延ばさず、表示範囲を左右または上下へ広げます。画面表示と出力に反映されます。
- **PDF用紙比率**：グラフのみPDFでは、PDFの用紙比率だけを1:1・4:3・16:9などに変更できます。この設定ではグラフ自体を引き延ばさず、余白を足して用紙比率を調整します。

## 9. 入力のコツ・エラー例

| 状況 | 対応 |
|---|---|
| 「数式を解釈できません」 | `*` の付け忘れや括弧の対応を確認（`(2x+1)(x-1)` など） |
| 「変数 x または y が必要です」 | 定数だけの式になっていないか確認 |
| 「未対応の変数」 | `a` などは使えません。`x`・`y`（媒介変数は `t`）のみ |
| 点の座標が反映されない | `x` や `y` などの変数は点座標には使えません。`pi`、`e`、数値、関数だけで書いてください |
| 「不等号の形式を確認」 | 不等号は1つ、範囲指定は向きをそろえて2つまで |
| 「定義域外の値」 | `sqrt` や `log` の定義域、または表示範囲を確認 |

> 補足：`sqrt(-1)` などは複素数ではなく「値なし（グラフを描かない）」として扱います。
