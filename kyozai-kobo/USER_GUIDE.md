# 教材工房 ユーザーガイド

このガイドは、教材工房で問題を作り、教材プロジェクトに並べ、LaTeX テンプレートから PDF を出力するまでの操作をまとめたものです。

テンプレートの書き方は後半の「LaTeX テンプレート詳説」に詳しくまとめています。

## 1. 教材工房でできること

教材工房は、Windows 上で動く LaTeX 教材作成アプリです。問題、解答、解説、画像、テンプレートをアプリ内で管理し、問題冊子、解答冊子、合本 PDF を生成できます。

主な画面は次の通りです。

| 画面 | 役割 |
| --- | --- |
| 問題バンク | 問題文、解答、解説、タグ、難易度、添付画像を管理します。 |
| 教材 | 問題バンクから問題を選び、章、節、説明文、部品と一緒に並べて冊子を作ります。 |
| 部品 | 注意書き、公式まとめ、画像ブロック、改ページなど、複数教材で使い回す素材を管理します。 |
| テンプレート | PDF 出力に使う LaTeX テンプレートを作成、編集、テストします。 |
| 設定 | TeX Live / MiKTeX の検出、出力先、バックアップなどを設定します。 |

データはローカル PC に保存されます。クラウドログインなしで使えます。

## 2. 初回設定

PDF を生成するには、`uplatex` と `dvipdfmx` が必要です。TeX Live または MiKTeX をインストールしたうえで、アプリの「設定」画面から TeX コマンドを検出してください。

基本の流れは次の通りです。

1. TeX Live または MiKTeX をインストールします。
2. 教材工房を起動します。
3. 「設定」を開き、「TeX コマンドを自動検出」を実行します。
4. 検出されない場合は、`uplatex.exe` と `dvipdfmx.exe` の場所を手動で指定します。
5. テンプレート画面または問題編集画面でプレビューを実行し、PDF が生成できるか確認します。

## 3. 問題バンク

問題バンクでは、あとで教材に組み込む問題を管理します。

### 3.1 分類

問題は、科目、分類、単元の階層で整理します。たとえば「数学」「数I」「二次関数」のように分類しておくと、検索や教材への追加が楽になります。

分類はあとから変更できます。問題一覧で複数選択して、別の単元へ移動することもできます。

### 3.2 問題の編集

問題編集画面では、主に次の項目を入力します。

| 項目 | 内容 |
| --- | --- |
| タイトル | 問題一覧や検索で見つけやすくする名前です。PDF の問題番号には通常使われません。 |
| 問題文 | 問題冊子に出力される LaTeX 本文です。 |
| 解答 | 解答冊子に出力される解答です。 |
| 解説 | 出力設定で「解説を含める」をオンにしたとき、解答冊子に出力されます。 |
| 難易度 | 基礎、標準、発展などの管理用の難易度です。 |
| A/B/C/D | PDF 上に表示できる小さなランクです。 |
| ★最低限 | PDF 上に `★` を表示できる必須問題フラグです。 |
| タグ | 検索や絞り込みに使います。 |
| メモ | 管理用のメモです。通常 PDF には出ません。 |

問題文、解答、解説は LaTeX としてそのまま挿入されます。数式は `$...$`、別行立ては `\[...\]` など、通常の LaTeX 記法を使えます。

### 3.3 添付画像

問題には PNG、JPG、PDF を添付できます。添付したファイルはアプリ管理下にコピーされ、PDF 出力時に一緒に使われます。

画像を問題文に入れるときは、エディタの画像挿入ボタンを使うか、LaTeX で次のように書きます。

```tex
\begin{center}
  \includegraphics[width=0.6\linewidth]{画像ファイル名.png}
\end{center}
```

画像を使うテンプレートには、通常次のパッケージが必要です。

```tex
\usepackage[dvipdfmx]{graphicx}
```

標準テンプレートには最初から入っています。

### 3.4 グラフ挿入

グラフ作成アプリとの連携が有効な場合は、問題文、解答、解説にグラフ画像を挿入できます。挿入された画像も添付ファイルとして保存されます。

### 3.5 履歴

問題を保存すると履歴が残ります。以前の内容に戻したい場合は、「履歴」から復元できます。復元時も現在の内容は履歴として保存されるため、戻し直すことができます。

## 4. 部品ライブラリ

部品は、複数の教材で使い回す短い素材です。たとえば次のようなものに向いています。

- 授業冒頭の注意書き
- 公式まとめ
- 図や表の共通ブロック
- 解答冊子だけに出したい補足
- 改ページ

部品にもタグ、表示先、履歴があります。教材に追加した時点の内容がスナップショットとして保存されるため、あとから部品本体を変更しても、すでに作った教材の出力は勝手には変わりません。

## 5. 教材プロジェクト

教材プロジェクトでは、問題バンクや部品を並べて冊子を作ります。

### 5.1 教材に追加できる項目

| 項目 | 用途 |
| --- | --- |
| 問題 | 問題バンクから追加します。問題文、解答、解説、添付ファイルがスナップショット保存されます。 |
| 章見出し | `section` 相当の見出しです。目次にも反映できます。 |
| 節見出し | `subsection` 相当の見出しです。 |
| 説明文 | 教材中に直接入れる LaTeX 本文です。 |
| 部品 | 部品ライブラリから追加します。 |
| 改ページ | PDF 出力時に `\newpage` を挿入します。 |

項目はドラッグして並べ替えできます。

### 5.2 スナップショット

教材に問題や部品を追加すると、その時点の内容が教材内に保存されます。これをスナップショットと呼びます。

スナップショットがあるため、問題バンク側の問題をあとから編集しても、既存教材の PDF は自動では変わりません。教材側に「最新版に更新」のような表示が出た場合だけ、必要に応じて更新します。

この仕組みにより、過去に作った教材を同じ見た目で再出力しやすくなります。

### 5.3 テンプレートのスナップショット

教材プロジェクトは、使用しているテンプレートの内容もスナップショット保存します。テンプレート本体をあとから編集しても、既存教材の出力はすぐには変わりません。

PDF 出力時にテンプレート本体が更新されている場合は、最新版に更新して出力するか、保存済みスナップショットのまま出力するかを選べます。

## 6. 出力設定

教材プロジェクトの「出力設定」では、PDF に出すタイトル、ヘッダー、番号、解答冊子の構成などを設定します。

### 6.1 基本項目

| 設定 | 主な影響 |
| --- | --- |
| 教材タイトル | `{{TITLE}}` に入ります。未入力ならプロジェクト名が使われます。 |
| 副題 | `{{SUBTITLE}}` に入ります。 |
| 学年・対象 | `{{TARGET}}` に入ります。氏名欄の行にも表示されます。 |
| 日付 | `{{DATE}}` に入ります。ヘッダー右の既定値にもなります。 |
| ヘッダー左 | `{{HEADER_LEFT}}` に入ります。未入力なら教材タイトルです。 |
| ヘッダー右 | `{{HEADER_RIGHT}}` に入ります。未入力なら日付です。 |
| 問題番号の形式 | `問題{n}`、`第{n}問` などを指定します。`{n}` が連番に置き換わります。 |

### 6.2 表示オン・オフ

| 設定 | オンのとき |
| --- | --- |
| 教材タイトルを表示する | `{{TITLE}}`、`{{SUBTITLE}}`、`{{ANSWER_TITLE}}` が出ます。オフなら空になります。 |
| ヘッダーを表示する | `{{HEADER_LEFT}}`、`{{HEADER_RIGHT}}` が出ます。オフなら空になり、`fancyhdr` のヘッダー線も消します。 |
| 氏名欄を表示する | 問題冊子、合本の `{{NAME_FIELD}}` に氏名欄が出ます。解答冊子には出ません。 |
| 問題番号を自動で振る | 各問題に `問題1` のような見出しを付けます。 |
| 問題ごとに改ページする | 各問題のあとに `\newpage` を挿入します。 |
| 目次を付ける | `\tableofcontents` を挿入します。テンプレートに `{{TOC}}` があればその位置、なければ本文先頭です。 |
| 章・節見出しに番号を振る | 章、節見出しを番号付きにします。 |
| 章ごとに問題番号をリセット | 番号付き章では `2-1` のように章番号付きで問題番号を振ります。 |

### 6.3 解答冊子

| 設定 | 影響 |
| --- | --- |
| 解答冊子に問題文を含める | 解答の前に問題番号と問題文を出します。 |
| 解答冊子の問題文を枠で囲む | 問題文を `tcolorbox` の枠で囲みます。 |
| 解答冊子に解説を含める | 解答の後ろに解説を出します。 |
| 解答冊子の2段組 | 解答冊子全体、または解答部分だけを `multicols` で2段組にできます。 |

解答冊子の最初に出る「タイトル 解答」を消したい場合は、「教材タイトルを表示する」をオフにします。新しいテンプレートでは `{{ANSWER_TITLE}}` を使うと、このオン・オフに自然に連動します。

### 6.4 難易度と星

問題編集画面で設定した A/B/C/D と `★最低限` は、出力設定で表示できます。

| 設定 | 出力例 |
| --- | --- |
| A/B/C/D 表示なし | `問題1` |
| 問題番号の横に表示 | `問題1 ★A` |
| 問題の右側に表示 | 問題見出しの右側に `★A` を寄せて表示 |
| ★ 表示なし | A/B/C/D のみ表示 |
| 最低限問題だけ表示 | `★` フラグがある問題だけ星を表示 |

`★` と A/B/C/D は、問題番号の文字列自体をずらさず、番号の近くに小さく表示されます。

## 7. PDF と .tex の出力

教材画面から次の出力ができます。

| 出力 | 内容 |
| --- | --- |
| 問題冊子 PDF | 問題文、章、節、説明文、部品を出力します。 |
| 解答冊子 PDF | 解答、必要に応じて問題文と解説を出力します。 |
| 合本 PDF | 問題冊子の後ろに解答編を付けた PDF を出力します。 |
| .tex プレビュー | 生成される LaTeX ソースを確認します。 |
| .tex 書き出し | 生成済み `.tex` と必要な画像を出力先に保存します。 |

PDF 生成に失敗した場合は、エラーメッセージと LaTeX ログの該当行を確認します。まずは `.tex プレビュー` で実際に生成されたソースを見ると原因を探しやすくなります。

## 8. LaTeX テンプレート詳説

ここからはテンプレートを書く方向けの詳しい説明です。

### 8.1 テンプレートの基本

テンプレートは、LaTeX 文書全体のひな形です。アプリはテンプレート内の `{{TITLE}}` や `{{BODY}}` などのプレースホルダーを、教材の内容に置換して `.tex` を生成します。

テンプレートは「テンプレート」画面で管理します。標準テンプレートは `uplatex + dvipdfmx`、A4、`ujarticle` を想定しています。

テンプレートには次の3つの入力欄があります。

| タブ | 用途 |
| --- | --- |
| 問題冊子用 | 問題冊子と合本の土台になります。 |
| 解答冊子用 | 解答冊子の土台になります。 |
| 共通（本文用） | 問題冊子用、解答冊子用が空のときに使われる予備テンプレートです。 |

通常は「問題冊子用」と「解答冊子用」を別々に書くのがおすすめです。

### 8.2 最小の問題冊子テンプレート

問題冊子用テンプレートには、少なくとも `{{BODY}}` または `% APP_BODY_START` / `% APP_BODY_END` が必要です。

```tex
\documentclass[uplatex,a4paper,11pt]{ujarticle}
\usepackage[dvipdfmx]{graphicx}
\usepackage{amsmath,amssymb,mathtools}

\begin{document}

\begin{center}
  {\LARGE \bfseries {{TITLE}}\par}
  {\large {{SUBTITLE}}\par}
\end{center}

{{NAME_FIELD}}

{{BODY}}

\end{document}
```

`{{BODY}}` の位置に、章、節、説明文、問題文、部品が入ります。

### 8.3 最小の解答冊子テンプレート

解答冊子用テンプレートには、通常 `{{ANSWER_BODY}}` を入れます。

```tex
\documentclass[uplatex,a4paper,11pt]{ujarticle}
\usepackage[dvipdfmx]{graphicx}
\usepackage{amsmath,amssymb,mathtools}

\begin{document}

\begin{center}
  {\LARGE \bfseries {{ANSWER_TITLE}}\par}
  {\large {{SUBTITLE}}\par}
\end{center}

{{ANSWER_BODY}}

\end{document}
```

`{{ANSWER_TITLE}}` は、出力設定で教材タイトルを表示する場合だけ `教材タイトル　解答` に置き換わります。タイトルを非表示にしたい運用では、解答冊子テンプレートに `{{TITLE}} 解答` と直接書くより、`{{ANSWER_TITLE}}` を使うほうが安全です。

### 8.4 プレースホルダー一覧

| プレースホルダー | 置換内容 |
| --- | --- |
| `{{TITLE}}` | 教材タイトルです。未入力ならプロジェクト名です。タイトル表示オフなら空です。 |
| `{{ANSWER_TITLE}}` | 解答冊子用のタイトルです。通常は `教材タイトル　解答` です。タイトル表示オフなら空です。 |
| `{{SUBTITLE}}` | 副題です。タイトル表示オフなら空です。 |
| `{{TARGET}}` | 学年、対象です。 |
| `{{DATE}}` | 日付です。 |
| `{{NAME_FIELD}}` | 学年、日付、氏名欄をまとめた行です。解答冊子では氏名欄は出ません。 |
| `{{HEADER_LEFT}}` | ヘッダー左です。未入力なら教材タイトルです。ヘッダー表示オフなら空です。 |
| `{{HEADER_RIGHT}}` | ヘッダー右です。未入力なら日付です。ヘッダー表示オフなら空です。 |
| `{{BODY}}` | 問題冊子、合本では問題本文全体です。解答冊子では `{{ANSWER_BODY}}` がない場合の代替として解答本文が入ります。 |
| `{{ANSWER_BODY}}` | 解答冊子の本文です。問題見出し、解答、必要に応じて問題文と解説が入ります。 |
| `{{EXPLANATION_BODY}}` | 解答冊子で解説だけを別の場所に出したい場合に使います。 |
| `{{PAGE_BREAK}}` | `\newpage` に置き換わります。 |
| `{{TOC}}` | 目次表示オンなら `\tableofcontents`、オフなら空です。 |

大文字の `{{SOMETHING}}` のような未定義プレースホルダーを書くと、保存時に警告が出ます。未定義プレースホルダーは置換されません。

### 8.5 `{{BODY}}` と `{{ANSWER_BODY}}` の使い分け

問題冊子用テンプレートでは、基本的に `{{BODY}}` を使います。

解答冊子用テンプレートでは、基本的に `{{ANSWER_BODY}}` を使います。

解答冊子用テンプレートに `{{ANSWER_BODY}}` と `{{BODY}}` の両方がある場合、`{{ANSWER_BODY}}` には解答本文、`{{BODY}}` には問題冊子側の本文が入ります。問題と解答を特殊な配置で並べたい場合に使えますが、通常の解答冊子では `{{ANSWER_BODY}}` だけで十分です。

### 8.6 解説を別ページ、別位置に出す

解答冊子用テンプレートに `{{EXPLANATION_BODY}}` を置くと、`{{ANSWER_BODY}}` には解説を含まない解答本文が入り、`{{EXPLANATION_BODY}}` に解説だけが入ります。

例:

```tex
{{ANSWER_BODY}}

\clearpage
\section*{解説}
{{EXPLANATION_BODY}}
```

解説を各解答の直後に出したい場合は、`{{EXPLANATION_BODY}}` を使わず、出力設定の「解答冊子に解説を含める」をオンにします。

### 8.7 目次の位置を指定する

出力設定で「目次を付ける」をオンにした場合、テンプレートに `{{TOC}}` があればその位置に目次が入ります。

```tex
\begin{document}

{{TOC}}

\clearpage

{{BODY}}

\end{document}
```

テンプレートに `{{TOC}}` がない場合は、アプリが本文の先頭に `\tableofcontents` を挿入します。

章、節を目次に載せるには、教材プロジェクト側で章見出し、節見出しを追加してください。

### 8.8 APP_BODY マーカー

既存の `.tex` をテンプレート化するとき、`{{BODY}}` を入れにくい場合は、次のコメントマーカーを使えます。

```tex
% APP_BODY_START
ここに書いた内容は出力時にアプリ生成本文で置き換わります。
% APP_BODY_END
```

アプリは `% APP_BODY_START` と `% APP_BODY_END` の間に本文を挿入します。マーカー間の元の内容は残りません。

通常は `{{BODY}}` または `{{ANSWER_BODY}}` を使うほうが分かりやすいです。マーカーと `{{BODY}}` を同時に置くと本文が重複することがあるため、基本的にはどちらか一方にしてください。

### 8.9 パッケージの書き方

テンプレート画面の「必要パッケージのメモ」は記録用です。PDF 出力には影響しません。

実際にパッケージを使うには、テンプレート本文のプリアンブルに `\usepackage` を書きます。

```tex
\usepackage[dvipdfmx]{graphicx}
\usepackage{amsmath,amssymb,mathtools}
\usepackage{geometry}
\usepackage{fancyhdr}
\usepackage{multicol}
\usepackage{tcolorbox}
\tcbuselibrary{skins}
```

よく使うパッケージは次の通りです。

| パッケージ | 用途 |
| --- | --- |
| `graphicx` | 画像挿入です。`dvipdfmx` オプション推奨です。 |
| `amsmath`, `amssymb`, `mathtools` | 数式です。 |
| `geometry` | 余白設定です。 |
| `fancyhdr` | ヘッダー、フッターです。 |
| `multicol` | 解答冊子の2段組です。 |
| `tcolorbox` | 解答冊子の問題文枠などです。 |
| `enumitem` | 箇条書き、番号付きリストの調整です。 |
| `float` | 図表位置の調整です。 |

解答冊子で「問題文を枠で囲む」を使う場合、アプリが必要に応じて `tcolorbox` と `skins` ライブラリを補います。ただし、自分で `tcolorbox` を使うテンプレートでは、プリアンブルに明示しておくのがおすすめです。

### 8.10 テンプレート用アセット

テンプレートにロゴ画像や `.sty` ファイルを持たせたい場合は、テンプレート画面の「テンプレート用アセット」に追加します。追加したファイルはコンパイル時にコピーされます。

ロゴ画像の例:

```tex
\usepackage[dvipdfmx]{graphicx}

\begin{document}
\noindent\includegraphics[width=30mm]{logo.png}

{{BODY}}
\end{document}
```

独自 `.sty` の例:

```tex
\usepackage{my-kyozai-style}
```

ファイル名が `my-kyozai-style.sty` の場合、LaTeX では拡張子なしで読み込みます。

### 8.11 ヘッダー、フッター

標準テンプレートは `fancyhdr` を使います。

```tex
\pagestyle{fancy}
\fancyhf{}
\lhead{{{HEADER_LEFT}}}
\rhead{{{HEADER_RIGHT}}}
\cfoot{\thepage}
\setlength{\headheight}{15pt}
```

`{{HEADER_LEFT}}`、`{{HEADER_RIGHT}}` は出力設定に連動します。ヘッダー表示をオフにすると空になり、`fancyhdr` を使っている場合はヘッダー線も消します。

`lhead` のように LaTeX の引数の中でプレースホルダーを使う場合は、標準テンプレートのように波かっこが3つ並ぶ形になります。

```tex
\lhead{{{HEADER_LEFT}}}
```

これは外側の `{...}` が LaTeX の引数、内側の `{{HEADER_LEFT}}` がアプリのプレースホルダーです。

### 8.12 タイトルまわり

問題冊子では `{{TITLE}}` を使います。

```tex
\begin{center}
  {\LARGE \bfseries {{TITLE}}\par}
  \vspace{2mm}
  {\large {{SUBTITLE}}\par}
\end{center}
```

解答冊子では `{{ANSWER_TITLE}}` を使います。

```tex
\begin{center}
  {\LARGE \bfseries {{ANSWER_TITLE}}\par}
\end{center}
```

出力設定で「教材タイトルを表示する」をオフにすると、`{{TITLE}}`、`{{SUBTITLE}}`、`{{ANSWER_TITLE}}` は空になります。

### 8.13 問題番号

問題番号はテンプレートではなく、アプリが本文を作る段階で生成します。

出力設定の「問題番号の形式」に `問題{n}` と書くと、`問題1`、`問題2` のように出ます。`第{n}問` と書けば、`第1問` になります。

章ごとのリセットがオンで、番号付き章の中にある問題は、`2-1` のように章番号付きになります。この場合、`問題{n}` は `問題2-1` になります。

A/B/C/D と `★` は、出力設定に応じて問題番号の近く、または問題見出しの右側に小さく入ります。

### 8.14 解答冊子の問題文枠

出力設定で次の2つをオンにすると、解答冊子で各問題文を枠で囲みます。

- 解答冊子に問題文を含める
- 解答冊子の問題文を枠で囲む

枠はアプリが `tcolorbox` で生成します。問題番号がタイトル位置に入り、問題文が枠の中に入ります。

テンプレート側では、通常 `{{ANSWER_BODY}}` を置くだけでかまいません。

```tex
{{ANSWER_BODY}}
```

枠を使いたくない場合は、「解答冊子の問題文を枠で囲む」をオフにしてください。問題文自体も不要な場合は、「解答冊子に問題文を含める」をオフにします。

### 8.15 2段組

解答冊子の2段組は、アプリが `multicols` 環境で本文を包みます。

テンプレートには次を入れておくと安心です。

```tex
\usepackage{multicol}
```

出力設定には次の選択肢があります。

| 設定 | 内容 |
| --- | --- |
| 2段組にしない | 通常の1段組です。 |
| 解答冊子全体を2段組 | 解答冊子本文全体を2段組にします。 |
| 解答部分だけを2段組 | 問題文は1段のまま、解答ブロックだけを2段組にします。 |

長い式や横幅の広い図表がある場合は、2段組でははみ出しやすくなります。その場合は1段組にするか、図の幅を `\linewidth` に合わせて調整してください。

### 8.16 既存 .tex の取り込み

テンプレート画面では、既存の `.tex` を取り込めます。

取り込み時の考え方は次の通りです。

| モード | 向いている場面 |
| --- | --- |
| そのまま使う | すでに `{{BODY}}` や APP_BODY マーカーを入れてある `.tex` を取り込む場合 |
| 本文を置換する | 既存の `\begin{document}` から `\end{document}` までをアプリ本文に置き換えたい場合 |

取り込んだら、テンプレート画面の「プレビュー」でサンプルデータをコンパイルして確認してください。

### 8.17 テンプレートの保存、履歴、複製

テンプレートは保存時に構文チェックされます。問題本文を挿入する場所がない場合や、不明なプレースホルダーがある場合は警告が出ます。

テンプレートにも履歴があります。変更して PDF が崩れた場合は、履歴から前の内容に戻せます。

別デザインを試したいときは、既存テンプレートを複製してから編集するのがおすすめです。

### 8.18 テンプレートのエクスポート、インポート

テンプレートは JSON としてエクスポートできます。別 PC に移したい場合やバックアップしたい場合に使います。

インポートすると、テンプレート本文、説明、パッケージメモ、アセット情報が読み込まれます。

## 9. よくあるトラブル

### PDF が生成できない

まず設定画面で `uplatex` と `dvipdfmx` が検出されているか確認してください。

次に `.tex プレビュー` を開き、エラーになっている行の近くを見ます。よくある原因は次の通りです。

- 必要な `\usepackage` がテンプレートにない
- 画像ファイル名が違う
- 数式の `$` や `{}` が閉じていない
- 2段組で図や表が横にはみ出している
- 既存テンプレートに `{{BODY}}`、`{{ANSWER_BODY}}`、APP_BODY マーカーがない

### テンプレートのメモにパッケージを書いたのに反映されない

「必要パッケージのメモ」は記録用です。実際に反映するには、テンプレート本文のプリアンブルに `\usepackage{...}` を書いてください。

### 解答冊子のタイトルだけ消したい

出力設定の「教材タイトルを表示する」をオフにしてください。解答冊子テンプレートでは `{{ANSWER_TITLE}}` を使うと、タイトル表示オン・オフに連動します。

### 解答冊子に問題文が出ない

出力設定の「解答冊子に問題文を含める」をオンにしてください。枠も付けたい場合は「解答冊子の問題文を枠で囲む」もオンにします。

### 目次の位置を変えたい

テンプレート内の好きな位置に `{{TOC}}` を置いてください。目次表示がオンならそこに `\tableofcontents` が入ります。

### 問題や部品を直したのに教材に反映されない

教材は追加時点のスナップショットを持っています。教材側で最新版に更新してください。

### テンプレートを直したのに教材に反映されない

教材はテンプレートもスナップショット保存しています。PDF 出力時に「最新版に更新して出力」を選ぶか、教材の使用テンプレートを更新してください。
